# Installation Guide

This guide explains how to install and run an OG Storage Node on Ubuntu.

## System Requirements

- Ubuntu 22.04 LTS
- 4 CPU cores
- 16 GB RAM
- 200 GB SSD
- Docker
- Git

## Step 1: Update the system

```bash
sudo apt update && sudo apt upgrade -y
```

## Step 2: Install Docker

```bash
curl -fsSL https://get.docker.com | bash
```

## Step 3: Clone the repository

```bash
git clone https://github.com/0glabs/0g-storage-node.git
```

## Step 4: Configure the node

Edit the configuration file according to your environment.

## Step 5: Start the node

Start the service and verify that it is running correctly.

## Verify

Check the node logs and ensure it begins syncing successfully.
